📘 SYNARCH — ФОРМАТ 2: “СПЕЦИФИКАЦИЯ”
Полный технический стандарт языка SynArch
🌑 РАЗДЕЛ 1. ОБЩЕЕ ОПИСАНИЕ
SynArch — это декларативный язык описания:

состояний,

связей,

весов,

логики,

памяти,

циклов,

условий,

системных операций.

Язык состоит из двух уровней:

Высокий слой (формульный) — описывает смысловые конструкции.

Низкий слой (системный) — предоставляет доступ к железу.

Файлы языка имеют расширение:

Код
.syn
🌒 РАЗДЕЛ 2. ЛЕКСИЧЕСКИЕ ЭЛЕМЕНТЫ
2.1. Идентификаторы
Код
IDENT ::= LETTER (LETTER | DIGIT | '_' )*
Примеры:

Код
Core
Values
psi
mu1
Layer_3
2.2. Числа
2.2.1. Обычные числа
Код
INT ::= DIGIT+
Примеры:

Код
1
5
12
256
2.2.2. Весовые числа
Код
WEIGHT ::= '@' DIGIT '.' DIGIT
Примеры:

Код
@0.1
@0.7
@1.0
2.3. Комментарии
Код
COMMENT ::= '#' ANY_TEXT
Пример:

Код
# это комментарий

🌘 РАЗДЕЛ 5. СИНТАКСИС ВЫРАЖЕНИЙ
5.1. Общая форма
Код
EXPR ::= ACTION [CONDITION]
5.2. Действие
Код
ACTION ::= TERM OP TERM
Где OP — один из:

Код
->  =>  =  +  -  v  ^
5.3. Условие
Код
CONDITION ::= 'if' EXPR
5.4. Примеры выражений
Код
psi(input) -> v
W(A->B) = @0.7
mu1 -> mu2 if weight > @0.3

🌕 РАЗДЕЛ 7. СИСТЕМНЫЙ СЛОЙ (НИЗКОУРОВНЕВЫЙ)
Системный слой предоставляет доступ к:

памяти,

устройствам,

портам,

регистрами,

потокам.

7.1. Доступ к памяти
Код
mem.read(ADDR)
mem.write(ADDR, INT)
Где:

Код
ADDR ::= '0x' HEX+
Пример:

Код
mem.read(0xFF01)
mem.write(0xFF02, 12)
7.2. Доступ к устройствам
Код
device.port(INT).send(EXPR)
device.port(INT).read()
7.3. Системные операции
Код
system.sleep(INT)
system.tick()
system.time()
🌑 РАЗДЕЛ 8. СТРУКТУРА ФАЙЛА .syn
Файл состоит из:

Код
FILE ::= (COMMENT | STATEMENT | EMPTY_LINE)*
Где STATEMENT — любое выражение из разделов 5–7.

Пример файла:

Код
# связи
W(A->B) = @0.8

# состояние
psi(input) -> v if W(values) < @0.3

# системный слой
mem.write(0xFF02, 12)
🌒 РАЗДЕЛ 9. ТРЕБОВАНИЯ К ИНТЕРПРЕТАТОРУ
Интерпретатор должен:

читать файл .syn,

разбирать лексемы,

строить AST (дерево выражений),

выполнять формулы высокого слоя,

выполнять команды системного слоя,

обеспечивать безопасность доступа к железу.
